# Email Reporting Engine

A production-grade frontend for an Email Reporting Analytics application. The project strictly adheres to WCAG 2.1 AA parameters and includes LLM-Assistant analytics capabilities built in.

## Features Included
- **Authentication**: JWT-mocked Role-Based Routing through Zustand.
- **Form Validations**: Deep validation structures handled by Zod and React Hook Form.
- **Analytics Dashboard**: High-performance dashboard utilizing custom Recharts visuals and Radix-based AI toggles.
- **Accessibility**: Comprehensive focus trapping, label pairing, ARIA describing, and screen reader announcements (`aria-live="polite"`).
- **Styling**: Tailwind CSS & Framer Motion.

## Tech Stack
- Next.js (App Router)
- React 19 + TypeScript
- Radix UI
- Zustand (Global Auth State)
- Tailwind CSS
- Zod + React Hook Form

## Local Deployment
1. Run `npm install`
2. Run `npm run dev`
3. Load `http://localhost:3000` to interact with Auth routing.

> Designed for rigorous visual adherence and state-of-the-art accessibility.
